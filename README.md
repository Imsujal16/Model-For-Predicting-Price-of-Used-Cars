# Model For Predicting Price of Used Cars

This project analyzes the CarDekho used-car dataset using Python. It explores the factors that influence resale price, including car age, kilometres driven, fuel type, transmission, and owner type.

## Project Contents

- `Dap_Mini_Project.ipynb` - Jupyter notebook containing data cleaning, exploratory data analysis, statistical testing, and K-Means clustering.
- `CAR DETAILS FROM CAR DEKHO.xls` - Source dataset used for the analysis.
- `miniProject_report_Sujal_Jaiswal_240662.pdf` - Mini project report summarizing the objective, methods, insights, and conclusion.

## Methods Used

- Data cleaning and duplicate handling
- Feature engineering using car age
- Label encoding for categorical variables
- Outlier handling using the IQR method
- Exploratory data analysis with visualizations
- One-way ANOVA to test the effect of fuel type on selling price
- K-Means clustering to group cars into market segments

## Key Insights

- Older cars and cars with higher kilometres driven generally fall into lower-price groups.
- Newer cars with lower kilometres driven tend to belong to premium segments.
- Fuel type has a statistically significant effect on selling price.
- Age and usage are strong indicators of resale value.
